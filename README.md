# Usage

to run
create & activate environment
```commandline
python -m venv .venv
```
MacOs / Linux
```commandline
source .venv/bin/activate 
```
Windows
```commandline
.venv/Scripts/activate
```

install packages
```commandline
pip install -r requirements.txt
```

go to `src` dir and run `train_models.py`

```commandline
cd src
python train_models.py
```
